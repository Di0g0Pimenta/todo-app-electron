# ToDo Calendar v0.1.0-alpha.3

Versao alfa para testes no Windows.

## Mudancas desde v0.1.0-alpha.1

- Corrigido o arranque da app instalada: a base de dados e o estado das notificacoes passam a ser guardados na pasta de dados do utilizador.
- A app passa a abrir diretamente em fullscreen.
- O executavel e os atalhos passam a usar o icon da app nas builds de release.
- Adicionado contador de tarefas pendentes no cabecalho.
- Adicionada pesquisa de tarefas na coluna lateral.
- Melhorada a validacao visual do campo obrigatorio "Nome" no modal de tarefa.
- Melhorado o script de dados demo com tarefas mais variadas, horas de aviso e estados concluidos.
- Atualizada a documentacao de releases para a versao alfa atual.

## Notas

- Esta build ainda e uma pre-release/alfa.
- O instalador Windows pode mostrar aviso de seguranca porque a app ainda nao esta assinada com certificado publico.
- Para testar notificacoes, cria uma tarefa com data de hoje e uma hora de aviso proxima.
